# fukuoka-todo-app

シンプルなTODOアプリケーションです。タスクに終了期限を追加できます。

## 機能

- タスクの追加・削除
- タスクの完了チェック
- 終了期限の設定（任意）
- 期限の状態表示（期限切れ、今日、今後）
- ローカルストレージでのデータ永続化

## 使い方

1. `index.html` をブラウザで開きます
2. タスク名を入力してタスクを追加します
3. 終了期限は任意で設定できます
4. チェックボックスでタスクの完了/未完了を切り替えられます
5. 削除ボタンでタスクを削除できます

## ファイル構成

- `index.html` - メインHTMLファイル
- `style.css` - スタイルシート
- `app.js` - アプリケーションロジック
